# TALENTLESS – Open Source Repository

There are two versions of TALENTLESS: an open-source version and a closed-source version.

The closed-source version is the full release of TALENTLESS. It contains all 500+ songs and is the version distributed and loaded by users.

The open-source version is intended as a keyless developer build. It is functionally similar to the full release for learning and modification, but includes a reduced public song set.

This repository contains all of the original scripts used in the real TALENTLESS. The only modification is that the song index in `MAIN.lua` includes a limited number of songs.

## Why does this repo exist?

To provide a transparent codebase that can be studied, modified, and extended without any key/license gate in this repository.

This repository exists so developers can study the structure and functionality of TALENTLESS in an open, editable form.

If you're interested in how TALENTLESS functions internally, this repository is for you.

Please make sure to ask for permission before using, modifying, or distributing this code.
